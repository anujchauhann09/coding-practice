# LeetCode SQL 50

This folder contains my solutions to the **LeetCode SQL 50** problems.

### About
SQL 50 is a curated list of SQL problems on LeetCode focusing on:
- SELECT queries
- JOINs
- Aggregations
- Subqueries

These problems help build practical SQL skills used in real-world data querying and analytics.

### Database Flavor
Solutions are written in standard SQL (MySQL / PostgreSQL compatible where possible).

### Goals
- Strengthen SQL querying skills
- Practice writing clean and efficient queries
