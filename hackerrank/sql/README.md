# HackerRank SQL

This folder contains my solutions to **SQL challenges from HackerRank**.

### About
This section focuses on:
- Basic SELECT statements
- Filtering and sorting
- Aggregations and GROUP BY
- JOINs and subqueries

These exercises help build a solid foundation in relational databases and querying.

### Database Flavor
Queries are written using standard SQL syntax (MySQL / PostgreSQL style).

### Goals
- Improve database querying skills
- Practice writing optimized and readable SQL
